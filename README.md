# Installation

The steps to install this extension are not too complex.

## Requirements

You need a Chromium-based browser (Chrome, Brave, Opera, etc.). Firefox is not yet supported.

## Download

Click the green **Code** button on this repo, select **Download ZIP**, then unzip the folder somewhere you'll remember.

## Install

Steps 2–4 differ slightly between browsers. Pick your section below.

### Brave

**1. Open the extensions menu (top right).**
<p><img width="537" alt="Brave extensions panel" src="https://github.com/user-attachments/assets/33fd729a-5396-4fca-ad06-09b27d719308" /></p>

**2. Enable Developer mode (top right of the extensions page).**

<img width="353" alt="Brave developer mode toggle" src="https://github.com/user-attachments/assets/d5905002-7500-42c8-a0de-22e7b5ed73dc" />

**3. Click "Load unpacked" (top left).**

<img width="608" alt="Brave load unpacked button" src="https://github.com/user-attachments/assets/3828cc10-f862-4acb-97ae-d34bf9e7f326" />

### Chrome

**1. Open the extensions menu (top right).**

<img width="772" alt="Chrome extensions button" src="https://github.com/user-attachments/assets/2b59b6cb-6aed-4667-8636-9f2270ec7f4f" />

**2. Enable Developer mode (top right of the extensions page).**

<img width="232" alt="Chrome developer mode toggle" src="https://github.com/user-attachments/assets/f51f794a-625a-45e4-9acf-c4641d246ffe" />

**3. Click "Load unpacked" (top left).**

<img width="621" alt="Chrome load unpacked button" src="https://github.com/user-attachments/assets/38cefe8f-be81-41d6-a9ff-b24b00a37659" />

### Final step (both browsers)

Select the unzipped folder. That's it.
